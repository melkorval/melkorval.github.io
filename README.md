# Small Multiplayer Video Game

## Project Description

This project is a small multiplayer video game where players can select a character and then move their character on a canvas map. The game uses an Express server with Node.js to manage player connections and interactions. Each player can see their own character move and interact with the game environment in real-time.

## Features

- Character selection
- Interactive canvas map
- Real-time player movements
- Backend server with Express and Node.js

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Additional Libraries: CORS
